# Ejercicio 2 - Funciones con parámetros

## Descripción
Programa en C# con 4 funciones:
- Suma de dos números enteros
- Verificación de mayoría/minoría de edad
- Área de un círculo (a partir del radio)
- Área de un triángulo (a partir de base y altura)

El programa Main() solicita los valores por teclado y muestra los resultados de cada función.
