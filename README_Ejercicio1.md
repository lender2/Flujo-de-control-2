# Ejercicio 1 - Análisis de números enteros

## Descripción
Programa en C# que permite introducir "n" cantidad de números enteros y determina:
- Cuántos son pares
- Cuántos son impares
- Cuántos son mayores a 10
- Cuántos son positivos
- Cuántos son negativos
